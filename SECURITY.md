# Security Policy

## Intended use

recon-kit-mcp performs network and security reconnaissance. Use it **only**
against assets you own or have explicit written permission to assess, for CTF
practice, or for education. You are responsible for how you use this software.

## Reporting a vulnerability

If you find a security issue in this project, please **do not open a public
issue**. Instead, report it privately via GitHub's
[Security Advisories](https://github.com/nan786521/recon-mcp/security/advisories/new)
for this repository.

Please include:

- A description of the issue and its impact
- Steps to reproduce (a minimal proof of concept if possible)
- Affected version(s)

You can expect an initial response within a few days. Once the issue is
confirmed and fixed, a new release will be published and the advisory disclosed.

## Supported versions

This project is pre-1.0; only the latest released version on
[PyPI](https://pypi.org/project/recon-kit-mcp/) is supported.
